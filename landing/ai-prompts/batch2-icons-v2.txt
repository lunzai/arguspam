UNIVERSAL RULES FOR ALL ICONS
===============================================

Keep It Simple:
- ONE main element per icon
- Maximum 2 elements if absolutely needed
- No complex details
- No readable text
- Soft shadows only
- Clean backgrounds

Style Consistency:
- Same rounded 3D style as hero
- Same lighting (soft, from top-left)
- Same shadow style
- Same color palette family
- Professional but friendly

Colors (ArgusPAM Brand):
- Blue: #3B82F6, #2563EB
- Green: #10B981, #7FDBCA
- Purple: #8B5CF6
- Orange: #F59E0B
- Red: #EF4444
- Gray: #6B7280

Avoid:
- Multiple objects
- Complex compositions
- Readable text
- Too many colors
- Cluttered designs
- Sharp edges
- Flat 2D style

Each icon should be instantly recognizable at small sizes!
